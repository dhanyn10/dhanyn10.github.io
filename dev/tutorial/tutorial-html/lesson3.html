Menggunakan Styles

Styles digunakan untuk membuat efek-efek css namun masih di dalam tag html. Untuk menerapkan styles, gunakan cara ini : <tagname style="nama_style:value_style;">...</tagname>

Coba untuk membuat warna teks menjadi biru pada tulisan di bawah ini dengan kode <p style="color:blue">text</p>

Selanjutnya, buat teks di tabel dibawah berwarna seperti ketentuan masing-masing

Teks Styles
merah menandakan berhenti atau bahaya color:red
orange menandakan bersiap atau siaga color:orange
hijau menandakan teruskan atau normal color:green
